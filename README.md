🧠 Breed Recognition using CNN

🚀 Overview
This project is a Deep Learning-based Breed Recognition system that classifies images into different categories using a Convolutional Neural Network (CNN).

The goal is to build an accurate and efficient model capable of identifying different breeds/classes from input images.
